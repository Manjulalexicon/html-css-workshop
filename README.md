
# Senior Java Backend Engineer Portfolio
### HTML & CSS Workshop Project

## Project Overview

This project is a professional single-page portfolio website built as part of an HTML & CSS workshop.

The purpose of this project is to demonstrate:

- Semantic HTML structure
- External CSS styling
- Layout design using Flexbox
- Table and form implementation
- Responsive design
- Proper Git workflow
- Deployment using GitHub Pages

The design follows a clean, enterprise-level corporate theme aligned with a backend engineering profile.

---

## Technologies Used

- HTML5
- CSS3
- Git
- GitHub
- GitHub Pages

---

## Workshop Requirements Implemented

- Semantic HTML elements (header, main, section, article, footer)
- External CSS file (styles.css)
- Technical skills table
- Contact form with labels and required validation
- Professional layout using Flexbox
- Responsive design (mobile optimized)
- Multiple meaningful Git commits
- GitHub Pages deployment
